If you see a blank page on GitHub Pages, it is because **GitHub Pages cannot run TypeScript (`.tsx`) files directly.** 

### The Solution: Deploy with Vercel
Vercel is built to handle React/TypeScript apps like this one.
1. Go to [Vercel.com](https://vercel.com).
2. Click **"Add New"** -> **"Project"**.
3. Import your GitHub repo `The-Golden-Jubilee`.
4. Under **Environment Variables**, add `API_KEY` (your Google Gemini Key).
5. Click **Deploy**. Vercel will build the code and give you a working link!
